= CppTest Utility
:toc: macro
:icons: font

**CppTest** is a powerful command-line tool for competitive programming. It automates compilation, static analysis, and testing of C++ solutions.

toc::[]

== ‚ú® Features

*   **Isolation:** By default, tests run in `/tmp`, keeping your project folder clean.
*   **Auto-Fix:** Flag `-F` automatically fixes formatting and simple linter errors.
*   **Comment Removal:** Flag `-R` strips all `//` and `/* */` comments before compiling.
*   **Static Analysis:** Automatically checks code with `clang-tidy` and `clang-format`.
*   **Custom Checkers:** Supports `check.cpp` for problems with multiple correct answers.
*   **Directory Support:** Run `cpptest ./ProblemA/` directly.

== üõ† Usage

=== Flags

[cols="1,3"]
|===
| Flag | Description

| `-i`, `--in-place`
| Run tests directly in source directory.

| `-F`, `--fix`
| **Auto-Fix.** Applies `clang-format` and `clang-tidy` fixes to your code. Modifies the source file!

| `-R`, `--no-comments`
| **Remove Comments.** Creates a temporary copy of the source code, strips all comments (`//` and `/* */`), and tests that clean version.

| `-f`, `--fast`
| **Fast mode.** Skip linters.

| `-K`, `--force-checker`
| **Force Checker.** Fail if `check.cpp` is missing.

| `-k`, `--force-diff`
| **Force Diff.** Ignore `check.cpp` and use standard diff.

| `-s`, `--select`
| Select default compiler.

| `-u`, `--update`
| **Self-Update.** Installs the latest version from GitHub.
|===
